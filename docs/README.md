# MAGS Module Documentation

Welcome to the **MAGS Resource Management System - UBLC (MAGS)** documentation. These guides are written for **student reviewers**, **faculty testers**, and **client stakeholders** who need to understand each part of the application without reading the source code.

## How to use this documentation

1. Complete **[00 — Getting Started](00-getting-started.md)** to run the app on your computer.
2. Read modules in order, or jump to the area you are testing.
3. Use the **demo accounts** in each guide to log in with the correct role.
